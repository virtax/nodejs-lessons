## Description

[Nest](https://github.com/nestjs/nest) example application.
+ Docker
+ PostgreSQL
+ Pug render
+ Validation
+ Unit tests

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development mode
$ docker-compose up --build
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
